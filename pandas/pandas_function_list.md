# pandas 関数・メソッド一覧

## 🔹 データ構造
| 名前 | 概要 |
|------|------|
| `Series` | 一次元のラベル付き配列 |
| `DataFrame` | 二次元のラベル付き表形式データ |

## 🔹 データ読み書き
| 名前 | 概要 |
|------|------|
| `read_csv()` / `to_csv()` | CSVファイルの読み書き |
| `read_excel()` / `to_excel()` | Excelファイルの読み書き |
| `read_json()` / `to_json()` | JSON形式の読み書き |
| `read_sql()` / `to_sql()` | SQLデータベースとのやりとり |

## 🔹 データ確認・基本操作
| 名前 | 概要 |
|------|------|
| `head()`, `tail()` | データの先頭・末尾を表示 |
| `info()`, `describe()` | 概要・統計要約を表示 |
| `shape`, `columns`, `index` | データの構造情報取得 |
| `dtypes`, `isnull()`, `notnull()` | データ型・欠損値確認 |

## 🔹 データ抽出・加工
| 名前 | 概要 |
|------|------|
| `loc[]`, `iloc[]` | ラベル・インデックスで抽出 |
| `query()` | 条件による抽出 |
| `drop()`, `rename()` | 行列の削除・名前変更 |
| `sort_values()` | 値でソート |
| `apply()`, `map()`, `replace()` | 関数適用・置換処理 |

## 🔹 集計・統計処理
| 名前 | 概要 |
|------|------|
| `groupby()` | グループ化集計 |
| `mean()`, `sum()`, `count()` | 基本統計 |
| `agg()`, `transform()` | 複合統計処理 |
| `pivot_table()` | ピボットテーブル |

## 🔹 欠損値・重複処理
| 名前 | 概要 |
|------|------|
| `isna()`, `fillna()` | 欠損の確認・補完 |
| `dropna()` | 欠損値の削除 |
| `duplicated()`, `drop_duplicates()` | 重複行の検出・削除 |

## 🔹 時系列処理
| 名前 | 概要 |
|------|------|
| `to_datetime()` | 日時型に変換 |
| `resample()` | 時系列の再サンプリング |
| `rolling()` | 移動平均などのロール演算 |
| `shift()`, `diff()` | シフト・差分計算 |
